Basic branding ideas

The aim for this website is to ensure readability for visitors. Using an analogous color scheme, I can guide visitors focus using more saturated colors, rounded borders, size of text, and large images. By adding hover animations when users interact with crucial interactive elements like buttons or images, it can gain the most attention as it contrasts with surrounding static elements.
